
Voorbeelden van het gebruik van de server:

    ./pqserve1 loginfile 6066

Voor loginfile, zie README in ../pqbuild1

Het getal aan het eind is een poortnummer.

Als het programma is gestart zoals boven kun je er gebruik van maken
door met je webbrowser deze url te openen:

    http://127.0.0.1:6066/
